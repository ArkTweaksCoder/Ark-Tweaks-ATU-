# UI Shell Plan

## Goals
- Present a premium dashboard experience for Ark Optimize.
- Group optimization features by capability and licensing state.
- Leave room for future desktop rendering and live telemetry widgets.

## Structure
- Header: product title, status, and quick actions.
- Sidebar: navigation for Optimize, Profiles, Diagnostics, and License.
- Main body: cards for performance, gaming, diagnostics, and license sections.
- Footer: version, links, and support status.
