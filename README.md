# Ark Suite

Ark Suite is a commercial-grade Windows optimization ecosystem centered on Ark Optimize, with companion products Ark Ping and Ark Mouse & Keyboard.

## Product Vision
- Safe, reversible, logged, and documented optimizations.
- Modular desktop applications with shared licensing and backend services.
- Privacy-preserving telemetry and enterprise-grade security.

## Repository Layout
- ArkOptimize/: flagship desktop application and optimization engine.
- ArkPing/: networking optimizer companion app.
- ArkKBM/: input optimization companion app.
- Backend/: REST API and telemetry services.
- Licensing/: Go-based activation and license validation service.
- Shared/: shared .NET contracts and models.
- Website/: marketing portal, account center, documentation.
- Installer/: Inno Setup packaging assets.
- Documentation/: product and engineering documentation.
